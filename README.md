## ✨ Insane Features

### 🧠 Advanced Security Engine
- **Cryptographic Entropy**: Uses Python's `secrets` module for industry-standard randomization.
- **Mathematical Strength Analysis**: Live entropy calculation in Bits to quantify exactly how secure your key is.
- **Dynamic Strength Meter**: Real-time visual feedback (Weak/Medium/Strong) as you tune your security vector.

### 🎭 Cinematic UI/UX
- **Pure Crimson Aesthetic**: A bespoke 3-layer design featuring a `#050505` base, intense red accents, and soft `#EAEAEA` highlights.
- **Hacker Glitch Reveal**: Immersive "decode" animation every time a password is initialized.
- **Interactive Particle Trail**: A high-performance fire-particle system that tracks your cursor across the grid.
- **CRT Scanline Overlay**: Authentic terminal vibes with a moving scanline system and pulsing glass shadow effects.

### 📋 Enterprise Workflow
- **Secure Buffer Log**: View your last 5 generated keys in a localized history tray.
- **Privacy Toggles**: Mask/Unmask history items with a single click to prevent over-the-shoulder snooping.
- **One-Click Deployment**: Optimized for Render with pre-configured `Procfile` and `gunicorn` support.

---

## 🚀 Quick Technical Specs

- **Backend**: Flask (Python)
- **Frontend**: HTML5 / Vanilla CSS (Glassmorphism + 3D Utilities) / Vanilla JS
- **Security**: `secrets.choice` for non-deterministic generation.
- **Performance**: Optimized for 60fps animations and low-latency interaction.

## 🛠️ Installation & Local Dev

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ManoharK-777/Password_Generator.git
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the engine:**
   ```bash
   python app.py
   ```

4. **Initialize:** Open your browser to `http://127.0.0.1:5000`
